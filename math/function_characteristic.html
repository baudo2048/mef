Characteristic Function



In this moment, it's almost impossible to give a general definition of Characteristic function. I must collect all the Characteristic type of definition and contexts. I have very few intuitive idea of it.

So, let's start with Characteristic function of a set.

LET \(X\) be a set.
LET \(X \ne \emptyset \).
LET \(E \subset X \).
LET \(x \in X\).

Definition. The function \( \chi_{A} : E \to \{0,1\} \), given by \[ \chi_{A}(x) = \begin{cases} 1, & \text{if } x \in A,\\ 0, & \text{if } x \notin A. \end{cases} \] is called characteristic function of the set \(A\).
