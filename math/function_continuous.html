Continuous Function

There are many different definition for continuous function. Some of them depend on the space we consider, for example, continuous function on metric space, continuous function on topological space and so on.

Definition: Let \(f\) be defined in a neighborhood of \(x_0\). The function \(f\) is continuous at \(x_0\) provided \[ \lim_{x \to x_0}{f(x)}=f(x_0) \].
