Metric

Let \(X \ne \emptyset \) be a set.
A real function \(d: X \times X \to R \) is a metric if \( \forall{x, y, z} \in X \) it satisfies the following properties:
  1. \( d(x,y) \ge 0 \)
  2. \( d(x,y) = 0 \) iif \(x=y\)
  3. \( d(x,y) = d(y,x) \) simmetry
  4. \( d(x,y) \le d(x,z) + d(z,y) \) triangular inequality
